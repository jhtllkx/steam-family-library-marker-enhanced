# Steam 家庭库已有游戏标记（增强版）

基于 [Cliencer Goge](https://greasyfork.org/zh-CN/scripts/491817) 的原版脚本修改，原脚本地址：https://greasyfork.org/zh-CN/scripts/491817

## 相比原版 v1.32 的增强功能

1. **成员卡片展示** — 设置面板中显示家庭组成员头像、名称、贡献游戏数
2. **独占/共同贡献分析** — 点击成员查看详细贡献记录，区分独占贡献和共同贡献
3. **贡献类型饼图** — 可视化展示独占 vs 共同贡献比例
4. **月度购入趋势图** — 折线图展示月度购入趋势，支持 12/24/36/60 个月及全部时间范围切换
5. **成员详情查看** — 点击成员卡片或图表柱子均可查看该成员的详细贡献信息
6. **头像持久化** — 成员头像 hash 存储，避免重复请求

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 [steam-family-library-marker-enhanced.user.js](./steam-family-library-marker-enhanced.user.js) 查看源码
3. 点击 Raw 按钮或复制全部内容，在 Tampermonkey 中新建脚本粘贴
